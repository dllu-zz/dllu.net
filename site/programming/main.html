Salvation (js) (2014)

Top-down shooter game featuring visibility polygons, created for Global Game Jam 2014. The object is to navigate to the exit whilst avoiding enemies. The player has the option to shoot and kill or stun enemies they encounter.
Raindrops (js) (2013)

Refracting raindrops on a piece of glass based on Gaussian metaballs. The normals of the raindrops are the gradient of the Gaussians.
Variational path finder (js) (2013)

Finds a path from the top left corner to the bottom right whilst avoiding the points. The path is a polyline and is locally optimized by gradient descent from the points represented by Gaussians.
Voronoi Maze (js) (2013)

Maze generator using a Voronoi diagram generated from random points. The generator runs is randomized depth first search on the Delaunay triangulation graph.
Hexmaze (js) (2013)

Hexagonal maze generator. The shortest path from the player to the mouse cursor is shown as a rainbow-coloured road. The generator is randomized depth first search.
Bouncing atoms (js) (2013)

Simulator for atoms trapped in a harmonic potential.
Michelson Interferometer (js) (2013)

Simulator of a Michelson interferometer with a thin layer of benzene on the beam splitter.
Dynamical Systems (js) (2012)

A collection of dynamical systems (the swinging Atwood's machine; the double pendulum; the simple damped driven pendulum; the Toda oscillator; the Duffing oscillator) simulated using the Runge Kutta 4 method.
Monte Carlo Localisation (js) (2012)

A robot attempting to localise in a randomlly generated 2D world using a noisy laser rangefinder. The pose is estimated using a particle filter. The world is generated using metaballs.
Pistoncraft (js) (2012-)

An unfinished game where the objective is to use pistons to construct a factory to produce robots.
BML Traffic Model (js, c++) (2010-)

A simulator for the Biham-Middleton-Levine traffic model, a simple 2D, 3-state cellular automaton which demonstrates phase changes and self-organisation.
Maze (js) (2012)

Simple maze generator using a randomized depth first search.
Metaballs (js) (2012)

Metaballs which repel those of the opposite colour but are attracted to those of the same colour.
Conway Tone Matrix (Flash) (2011)

A tone matrix which also simulates Conway's game of life, thereby resulting in an unlimited variety of musical sounds. The game is also available on Blackberry Playbook.
RPSContest (python) (2011)

An analysis of algorithms that play rock-paper-scissors.
